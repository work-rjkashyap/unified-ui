# Support

## Getting Help

For setup and usage questions, start with:

- Repository README:
  https://github.com/imrj05/unified-ui#readme
- Package docs:
  https://www.npmjs.com/package/@work-rjkashyap/unified-ui

## Where To Ask

- Open a GitHub issue for reproducible bugs and actionable feature requests.
- Use discussions or maintainer contact for broader questions and feedback:
  https://github.com/imrj05

## Before Opening an Issue

Please check:

- Existing issues for duplicates
- Latest `main` branch behavior
- Minimal reproduction in a fresh project

This helps reduce triage time and speeds up resolution.
