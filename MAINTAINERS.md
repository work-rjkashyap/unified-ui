# Maintainers

## Current Maintainer Team

- Lead maintainer: [@imrj05](https://github.com/imrj05)

## Maintainer Responsibilities

- Review and triage incoming issues and pull requests.
- Keep contribution and policy docs up to date.
- Ensure release quality for `@work-rjkashyap/unified-ui`.
- Coordinate changelog and tagged releases.
- Moderate community interactions under the Code of Conduct.

## Response Targets

- New issue triage: within 5 business days
- New pull request first review: within 7 business days
- Security report initial acknowledgement: within 5 business days

## Becoming a Maintainer

New maintainers may be added by existing maintainers based on sustained,
high-quality contributions and consistent review/triage participation.

Criteria typically include:

- Meaningful code or documentation contributions over time
- Constructive and reliable code reviews
- Familiarity with release and changelog process
- Alignment with project standards and Code of Conduct
