## Overview

The `Badge` component is a compact inline label used for categorization, status indicators, and metadata. It supports 7 semantic variants, dot indicators, removable state, and icon support.

---

## Import

```tsx
import { Badge } from "@work-rjkashyap/unified-ui";
// or
import { Badge } from "@work-rjkashyap/unified-ui/components";
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "primary" \| "success" \| "warning" \| "danger" \| "info" \| "outline"` | `"default"` | Semantic variant controlling colors. |
| `size` | `"sm" \| "md"` | `"md"` | Size of the badge. |
| `dot` | `boolean` | `false` | Show a colored dot indicator before the text. |
| `icon` | `ReactNode` | — | Icon to display before the text. |
| `removable` | `boolean` | `false` | Show a dismiss (×) button. |
| `onRemove` | `(event: MouseEvent) => void` | — | Callback when the remove button is clicked. |
| `removeLabel` | `string` | `"Remove"` | Accessible label for the remove button. |
| `as` | `ElementType` | `"span"` | The HTML element or component to render as. |
| `className` | `string` | — | Additional CSS classes. |

## Accessibility

<Callout type="info">
  Color is never the sole means of conveying information. Always pair semantic colors with a descriptive text label.
</Callout>

- The dot indicator is decorative (`aria-hidden="true"`) — the text label conveys the meaning.
- The remove button includes a configurable `aria-label` (defaults to `"Remove"`).
- All semantic variant color combinations meet WCAG AA contrast ratios on their muted backgrounds.
- When using `as="button"` or `as="a"`, the badge gains interactive semantics automatically.
