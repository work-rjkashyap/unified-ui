## Overview

The `Label` component provides an accessible `<label>` element for form controls. It supports required indicators, descriptions, multiple sizes, and disabled states. Built on Radix UI's Label primitive for proper accessibility semantics.

---

## Import

```tsx
import { Label } from "@work-rjkashyap/unified-ui/components";
// or from barrel (aliased to avoid collision with Typography Label)
import { FormLabel } from "@work-rjkashyap/unified-ui";
```

## Accessibility

<Callout type="info">
	The required indicator is decorative (`aria-hidden`). Always set `aria-required="true"` on the associated input for screen reader support.
</Callout>

- **Semantic HTML** â€” Renders as a native `<label>` element via Radix UI's Label primitive.
- **htmlFor linkage** â€” Clicking the label focuses the associated input.
- **Required indicator** â€” Visual only; use `aria-required` on the control.
- **Disabled state** â€” Visual only; disable the input itself for functional disabling.
- **Peer-disabled** â€” Supports Tailwind's `peer-disabled` pattern for automatic styling when the input is disabled.
