## Overview

The `RadioGroup` component is used for selecting a single option from a set. It supports standard radio buttons, a card variant for richer selection UIs, descriptions, two sizes, horizontal/vertical orientation, and disabled states.

---

## Import

```tsx
import {
	RadioGroup,
	RadioGroupItem,
	RadioCard,
} from "@work-rjkashyap/unified-ui";
// or
import {
	RadioGroup,
	RadioGroupItem,
	RadioCard,
} from "@work-rjkashyap/unified-ui/components";
```

## Props

### RadioGroup

| Prop            | Type                         | Default      | Description                            |
| --------------- | ---------------------------- | ------------ | -------------------------------------- |
| `value`         | `string`                     | ‚Äî            | Controlled selected value.             |
| `defaultValue`  | `string`                     | ‚Äî            | Initial selected value (uncontrolled). |
| `onValueChange` | `(value: string) => void`    | ‚Äî            | Callback when the selection changes.   |
| `size`          | `"sm" \| "md"`               | `"md"`       | Size of all radio items in the group.  |
| `orientation`   | `"vertical" \| "horizontal"` | `"vertical"` | Layout direction of items.             |
| `disabled`      | `boolean`                    | `false`      | Disables all items in the group.       |
| `className`     | `string`                     | ‚Äî            | Additional CSS classes.                |

### RadioGroupItem

| Prop          | Type        | Default      | Description                             |
| ------------- | ----------- | ------------ | --------------------------------------- |
| `value`       | `string`    | **Required** | The value of this radio item.           |
| `label`       | `ReactNode` | ‚Äî            | Clickable label text.                   |
| `description` | `ReactNode` | ‚Äî            | Additional description below the label. |
| `disabled`    | `boolean`   | `false`      | Disables this individual item.          |
| `className`   | `string`    | ‚Äî            | Additional CSS classes.                 |

### RadioCard

| Prop          | Type        | Default      | Description                       |
| ------------- | ----------- | ------------ | --------------------------------- |
| `value`       | `string`    | **Required** | The value of this radio card.     |
| `label`       | `ReactNode` | ‚Äî            | Card title / label text.          |
| `description` | `ReactNode` | ‚Äî            | Description text below the label. |
| `disabled`    | `boolean`   | `false`      | Disables this card.               |
| `className`   | `string`    | ‚Äî            | Additional CSS classes.           |

## Accessibility

<Callout type="info">
	Built on **Radix UI's Radio Group** primitive for full keyboard and screen
	reader support out of the box.
</Callout>

- **Arrow keys** navigate between items within the group.
- **Space** selects the focused item.
- Each item has proper `role="radio"` and `aria-checked` attributes.
- Labels are associated with their radio inputs for click-to-select behavior.
- Focus ring is visible on keyboard navigation only (`focus-visible`).
- Disabled items are excluded from the tab order and marked with `aria-disabled`.
