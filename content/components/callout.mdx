## Import

```tsx
import { Callout } from "@work-rjkashyap/unified-ui";
// or
import { Callout } from "@work-rjkashyap/unified-ui/components";
```

## Basic Usage

Wrap any content in `Callout` with a `variant` and optional `title`. The default icon is an info circle, matching the variant's color.

```tsx
<Callout variant="info" title="Did you know?">
  Unified UI components are fully tree-shakeable via layer-specific imports.
</Callout>
```

## In Context

A common pattern is to place callouts above or below forms to provide contextual guidance.

```tsx
<form className="flex flex-col gap-4">
  <Callout variant="info" title="Changing your email">
    A verification link will be sent to your new address. Your current email
    remains active until the new one is confirmed.
  </Callout>

  <div className="flex flex-col gap-1.5">
    <label className="text-sm font-medium">New Email</label>
    <Input type="email" placeholder="you@example.com" />
  </div>

  <Button variant="primary">Update Email</Button>
</form>
```

## Props

| Prop          | Type                                                          | Default   | Description                                              |
| ------------- | ------------------------------------------------------------- | --------- | -------------------------------------------------------- |
| `variant`     | `"info" \| "success" \| "warning" \| "danger" \| "default"`  | `"info"`  | Semantic color variant.                                  |
| `title`       | `ReactNode`                                                   | —         | Optional heading rendered in bold above the body.        |
| `icon`        | `ReactNode \| null`                                           | —         | Custom icon. Pass `null` to hide the default icon.       |
| `collapsible` | `boolean`                                                     | `false`   | Whether the body can be toggled open/closed.             |
| `defaultOpen` | `boolean`                                                     | `true`    | Initial open state when `collapsible` is true.           |
| `className`   | `string`                                                      | —         | Additional CSS classes on the root element.              |
| `children`    | `ReactNode`                                                   | —         | The body content of the callout.                         |

## Accessibility

- The root element uses `role` semantics inherited from its semantic HTML.
- Collapsible header uses `role="button"`, `tabIndex={0}`, `aria-expanded`, and responds to `Enter` and `Space` key presses.
- The chevron icon is `aria-hidden="true"`.
- The default icon is `aria-hidden="true"`.
- Animated entrance respects `prefers-reduced-motion` — disables Framer Motion variants when reduced motion is preferred.
- The collapsible body uses `expandHeight` animation for smooth height transitions (skipped under `prefers-reduced-motion`).
